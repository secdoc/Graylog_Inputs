# graylog
Graylog Security Dashboards

The Security_Dashboard.json file contains the security dashboard configurations for the following input types:

 - DHCPD
 - Maltrail
 - NAXSI
 - Netflow
 - OPNsense
 - Pihole/Unbound DNS/DNSMasq
 - Suricata
 - Zenarmor
